What We Do

Smart Compost, Smarter Planet

We built a smart compost monitoring system using an ESP32, AI, and sensors. Our system keeps compost healthy and sends real-time alerts when something's wrong.

📡
ESP32 + Sensors

Our hardware core — an ESP32 microcontroller reads live moisture, temperature, and other compost data and streams it in real time.

🤖
AI-Powered Analysis

Artificial intelligence interprets sensor readings to detect unhealthy compost conditions before they become a problem.

📲
QR Code Scanning

Scan any waste item's QR code to instantly know whether it belongs in the compost — no guessing required.

💧
Moisture Monitoring

Real-time moisture tracking sends an alert if the compost is too dry or too wet, keeping it in the perfect zone.

🚨
Smart Alerts

Instant notifications fire if the wrong item is added or if conditions drift outside healthy parameters.

🌍
Real-World Impact

How Trash Titans 2026 Works

Trash Titans 2026 is a full AI-powered smart waste management platform that combines QR scanning, Edge Impulse AI, compost monitoring, servo motor automation, live dashboards, and smart alerts.

1

User Login

Users sign into the Trash Titans web app with a secure account.

2

QR Code Scan

The USB webcam scans a QR code attached to the waste item.

3

Database Detection

The JSON database identifies whether the waste is compost, recycle, garbage, or hazardous.

4

AI Compost Detection

The moisture sensor activates and Edge Impulse AI predicts healthy, dry, or too wet compost.

5

Servo Motor Sorting

The SG90 servo motor rotates the sorting flap automatically.

6

LED Feedback System

Green, yellow, and red LEDs display live compost conditions.

7

Live Dashboard

The dashboard updates instantly with waste statistics and AI predictions.

8

Email Alerts

Automatic email alerts are sent if compost becomes too dry, too wet, or hazardous waste is detected.

🧠
Arduino UNO Q 4GB

Main controller powering the entire smart waste management system.

📷
USB Webcam

Scans QR codes and identifies waste instantly.

💧
Moisture Sensor

Detects compost health and moisture conditions in real time.

⚙️
SG90 Servo Motor

Automatically rotates the waste sorting flap.

🤖
Edge Impulse AI

Predicts healthy, dry, or too wet compost conditions.

🚨
Smart Alerts

Sends email alerts and activates LEDs and buzzer warnings.
